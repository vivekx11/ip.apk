# Order Tracking Feature - User Guide

## Kaise Use Karein (How to Use)

### 1. Order Place Karna (Placing an Order)

1. **Shop Select Karein**: Shops tab se apni pasand ki shop choose karein
2. **Products Add Karein**: Products ko cart mein add karein
3. **Cart Check Karein**: Cart icon pe tap karein
4. **Place Order**: "Place Order" button pe tap karein
5. **Details Fill Karein**:
   - Apna naam enter karein
   - Special instructions (optional)
6. **Submit**: "Place Order" button pe tap karein

### 2. Order Placed Ke Baad (After Order Placement)

Order place hone ke baad aapko ek message dikhega:


**Do Options:**
- **OK**: Previous screen pe wapas jaayein
- **View Orders**: Apne orders dekhein

### 3. Order Status Check Karna (Checking Order Status)

**Bottom Navigation se:**
1. "Orders" tab pe tap karein
2. Do tabs dikhenge:
   - **Current**: Active orders (Pending/Accepted)
   - **History**: Past orders (Completed/Cancelled)

### 5. Order Details Dekhna (Viewing Order Details)

**Kisi bhi order pe tap karein to dekhein:**

1. **Status Header**: 
   - Large colored header with current status
   - Status message

2. **Shop Details**:
   - Shop name
   - Shop address

3. **Order Information**:
   - Order date and time
   - Customer name
   - Special notes (if any)
   - Accepted/Completed timestamps

4. **Pickup Code** (Accepted orders only):
   - Large 4-digit code
   - Copy button
   - Shop pe ye code dikhana hai

5. **Order Items**:
   - All products with images
   - Quantity and prices
   - Total amount

6. **Payment Info**:
   - Reminder: Payment at shop

## Tips

1. **Pickup Code Save Karein**: Screenshot le lein ya copy kar lein
2. **Shop Timing Check Karein**: Order pickup ke liye shop timing dekh lein
3. **Payment Ready Rakhein**: Cash ready rakhein shop pe
4. **Order Details Check Karein**: Pickup se pehle order items verify kar lein

## Troubleshooting

### Order Pending Bahut Time Se Hai
- Shop owner ko time lagta hai accept karne mein
- Agar bahut time ho gaya, shop ko call karein

### Pickup Code Nahi Dikh Raha
- Order status "Accepted" hona chahiye
- Order details screen refresh karein
- Internet connection check karein

### Order History Load Nahi Ho Raha
- Internet connection check karein
- Pull to refresh karein
- App restart karein

### Order Cancel Karna Hai
- Abhi user cancel feature nahi hai
- Shop owner ko directly contact karein
